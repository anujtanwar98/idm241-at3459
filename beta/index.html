Triggers

    On-Hover

  • Hover over menu items
  • Hover over categories of menu items

    • On-Click

  • Click on Track a package category of Tracking menu item

Rules

On-Hover

  • The pointer changes to a hand pointer
  • A yellow line animate under the menu items
  • If hover over to other menu items with one menu item category open, yellow line still animates
  • If hover over to other menu items with one menu item category open, the dropdown menu closes
  • If hover over to categories of menu items, >> arrow animates from right and text animates to the left
  • If you hover over category of menu items the color will change from grey color #595959 to black color: #000

Off-Hover

  • Hand pointer changes to regular pointer
  • Yellow Line animates out and category box disappears

On-Click

  • On click of Track a package category of Tracking box animates down
