The Overview

Micro-interactions are small animations that occur when a user interacts with a website or app. They are used to enhance the user experience and make the interface more intuitive. Context and Challenge

In my IDM-241 class, our project goal was to create or improve micro-interactions.

The project was divided into three phases Alpha, Beta and Final and every phase has a Build Version over a period of 10 weeks.


The reason I picked UPS website as when a user visit UPS website, and try to interact with the menu system they have to click every menu item in order to see categories of the menu items, which is very annoying and time consuming for the user and in order to close menu system the user they have to click outside on the page. Category options like Tracking a user wants to interact with it without loading any page and very quickly.


So I decided to make it more intuitive and user friendly by adding micro-interactions, which can help user to interact with the menu system in a more efficient way.Firstly I added a micro-interaction to the menu system rather than clicking on the menu item to see the categories of the menu item, the user can hover over the menu item and the categories will appear. Secondly I added a micro-interaction when a user hover over category items they can interact and know which category they are one and user can click track a package to open the tracking box without loading any page and the user can enter the tracking number and click on the track button to see the tracking results without loading any page. The user can close the tracking box by clicking on the close button and user can also close the tracking results box by clicking on the close button.


This was a individual project that I did. From designing the micro-interactions to coding the micro-interactions, I did everything by myself. More importantly, it was also for us to know/learn what made for a truly engaging interaction.

The Process and Insight

I looked different website to began with. I wanted to pick a website which is very commonly used in day to day task to find or search something. I looked at different websites like Amazon, eBay, UPS, USPS, FedEx, etc. I decided to pick UPS website because it is very commonly used by people to track their packages.


I wanted to keep website very simple and wanted to use UPS logo color for the main content of the website to show all different triggers, rules, feedback and loops & modes.
