UPS Micro-interactions

ALPHA

www.ups.com/us/en/Home.page

Triggers

* Hovering over menu items

* clicking on menu items to show category

Rules

On-Hover

* The pointer changes to a hand pointer

* A yellow line animate under the menu items

* If hover over to other menu items with one menu item category open, yellow line still animates

On Click

* Category of the menu items pulls down with quick pull down with fade in effect

* The background opacity goes dark with 50%

* If click to other menu items category pulls out with fade out effect

Off-Hover

* Hand pointer changes to regular pointer

* Yellow Line animates out
